# New-Logarithms

An interactive web application for learning and practicing logarithmic equations, conversions, and problem-solving.

## Overview

This project provides a structured learning platform with curated logarithm problems across multiple difficulty levels—from basic conversions to advanced JEE-level questions. Each problem includes step-by-step solutions to help students understand the concepts.

## Features

- 📚 **Problem Database** — 8+ problems spanning Easy → JEE difficulty levels
- 📖 **Step-by-Step Solutions** — Detailed breakdown of each problem
- 🎯 **Progressive Learning** — Problems organized by difficulty
- 🚀 **Easy Deployment** — Ready-to-run Node.js server

## Tech Stack

- **Backend:** Node.js (built-in HTTP server)
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Replit-ready

## Getting Started

### Prerequisites
- Node.js 20+ installed

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ChakravarthulaSaiTeja/New-Logarithms.git
cd New-Logarithms
```

2. Start the server:
```bash
node server.js
```

3. Open your browser and navigate to:
```
http://localhost:5000
```

## Problem Categories

### Easy
- Exponential ↔ Logarithmic conversions
- Basic logarithm evaluation

### Medium
- Logarithmic equation solving
- Multi-step problems with domain checks

### Hard
- Logarithmic inequalities
- Complex equation solving

### JEE Level
- Logarithm proofs (e.g., change of base)
- Exponential-logarithmic hybrid equations

## Usage

1. Navigate to `http://localhost:5000`
2. Browse problems by difficulty level
3. Click on a problem to view step-by-step solutions
4. Study the steps and understand the logic
